# BGM (background music)

Taruh file MP3 (looping) di folder ini dengan nama berikut:

- `login.mp3` — profile picker, looping, welcome feel
- `room.mp3` — kamar Daffa & Dio (dipakai bersama), looping
- `admin_home.mp3` — admin, light calm, looping

Selama file belum ada, AudioManager otomatis fallback ke melodi Web Audio
(oscillator) — tidak ada error.

Sumber gratis no-copyright: https://pixabay.com/music (search "children game",
"fun kids") atau https://suno.ai (AI generate, free tier).
